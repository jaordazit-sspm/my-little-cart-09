
# Tienda SSPM

tienda con catalogo de productos, carrito de compras, registro de usuarios. con rol de usaurio o administrador.


## API Reference
```http
src/integrations/lovable
src/integrations/supabase
```

#### CRUD productos

```http
  Lovable genera automáticamente endpoints:
```

| Acción | ndpoint     |
| :-------- | :------- |
| `Crear` | `POST /api/usuarios` |
| `Leer` | `GET /api/usuarios` |
| `Leer uno` | `GET /api/usuarios/:id` |
| `Actualizar` | `PUT /api/usuarios/:id` |
| `Eliminar` | `DELETE /api/usuarios/:id` |
